# Overview

This project is an **Advent of Code 2022 Solver** written in C#. It demonstrates my understanding of C# programming by solving puzzle challenges from the Advent of Code 2022 event (https://adventofcode.com/2022). The software is a console-based application that provides an interactive menu for running puzzle solutions, viewing statistics, and managing input data.

The purpose of creating this software was to learn C# fundamentals including object-oriented programming, inheritance, structures, file I/O operations, and modern C# syntax while solving interesting algorithmic puzzles. This project serves as my CSE 310 Module 1 submission demonstrating proficiency in C#.

**Key Features:**
- Solves Day 1 (Calorie Counting) and Day 2 (Rock Paper Scissors) puzzles
- Interactive menu-driven interface
- Reads puzzle input from text files
- Displays formatted results with statistics tracking
- Demonstrates all required C# programming concepts
