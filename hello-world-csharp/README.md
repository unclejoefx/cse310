# Hello World - C#

## Overview

This is a simple "Hello World" program written in C#. It demonstrates basic console output and serves as an introduction to C# programming for CSE 310.

## Description

This program prints a greeting message to the console, including:
- The classic "Hello World!" message
- A welcome message
- The current date
- An inspirational message for the semester

## Development Environment

- **Language:** C# (.NET)
- **IDE/Editor:** Visual Studio Code / Visual Studio
- **Runtime:** .NET SDK

## How to Run

To execute the program, navigate to the project directory and run:

```bash
dotnet run
```

Or compile and run manually:

```bash
dotnet build
dotnet run
```
