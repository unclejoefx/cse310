# Overview

As a software engineer, I am working to build a strong foundation in programming languages and development tools. This Hello World project marks the beginning of my journey in CSE 310, where I aim to expand my skills and create a personal software portfolio.

This software is a simple "Hello World" program written in C#. It demonstrates basic console output functionality including printing a greeting message, a welcome message, the current date, and an inspirational message for the semester.

My purpose for creating this software is to familiarize myself with C# programming, set up my development environment, and establish my GitHub repository for the course. This serves as the foundation for more complex projects throughout the semester.

[Software Demo Video](https://youtu.be/Rb_qCw_1xkg)

# Development Environment

I used Visual Studio Code as my code editor along with the .NET SDK to compile and run the program. The terminal/command line was used to execute `dotnet run` commands.

The programming language used is C# (.NET). C# is a modern, object-oriented programming language developed by Microsoft that runs on the .NET platform.
